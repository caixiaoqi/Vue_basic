Vue.extend是扩展实例的构造器，主要为Vue.component()服务

      let baseOptions = { 
      template: `
 {{ firstName }} {{ lastName }} aka {{ alias }}
`,
      data ()
       { 
         return { 
           firstName: '大漠', lastName: 'w3cplus', alias: '大漠_w3cplus'
           } 
        }
      }
      let baseExtend = Vue.extend(baseOptions);
      new baseExtend()
      new baseExtend().$mount('#app')
      使用起来好像和new Vue（）一样啊
但是给这个组件扩展构造器预设一些参数，主语是服务于Vue.component的
